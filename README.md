# Pax 1933 Starter

Browser grand-strategy sandbox prototype.

- Backend: Python + FastAPI
- Frontend: React + TypeScript + Vite + MapLibre GL JS
- Main map: CShapes-derived country-level GeoJSON for 1933
- Extra map data: Cartography V1 dynamic country labels, real Admin-1 region layer when source data is present, and province debug layer
- Order flow: command parser skeleton, no AI judge or combat system yet

## Backend

```powershell
cd backend
.\.venv\Scripts\activate
pip install -r requirements.txt
python scripts\generate_provinces_1933.py
python scripts\prepare_admin1_regions.py
uvicorn main:app --reload
```

Backend runs at:

```text
http://127.0.0.1:8000
```

Useful checks:

```text
http://127.0.0.1:8000/api/health
http://127.0.0.1:8000/api/map/1933
http://127.0.0.1:8000/api/map/1933/current-countries
http://127.0.0.1:8000/api/map/1933/current-labels
```

## Current Architecture

`/api/map/1933` returns the historical country-level CShapes map. This is the stable political map used for the main visual layer.

`/api/map/1933/current-labels` builds country labels dynamically from the current country geometry. The backend groups current provinces by `ownerTag`, builds country geometry, and generates label lines/points from that geometry. After ownership changes, labels can be recalculated with the new borders.

`backend/data/base/provinces_1933.geojson` is an MVP province dataset for debugging and future gameplay. It is not the final province map and is hidden by default in the frontend. The `Show provinces` button enables only thin province borders, not province fill.

`backend/data/base/regions_1933.geojson` is the player-facing region border layer. It is generated by `backend/scripts/prepare_admin1_regions.py` from a real Admin-1 GeoJSON source such as Natural Earth Admin-1 States/Provinces or geoBoundaries ADM1, clipped against the 1933 CShapes country polygons.

The script uses `backend/data/raw/ne_10m_admin_1_states_provinces.geojson` by default. If it is missing, it tries to auto-download it from Natural Earth's public GitHub mirror. You can also place one of these files in `backend/data/raw/` manually:

```text
ne_10m_admin_1_states_provinces.geojson
ne_50m_admin_1_states_provinces.geojson
geoBoundaries_ADM1.geojson
```

If the raw file is missing and auto-download fails, the script exits with a clear error. The API can still return an empty safe layer, but successful Cartography V1 preparation should produce non-empty `regions_1933.geojson` and `regions_1933.json`. Pax 1933 does not render rough bbox rectangles as region borders.

`backend/scripts/prepare_cshapes_1933.py` can also merge real missing map units such as Greenland when a Natural Earth Admin-0/map units GeoJSON is present in `backend/data/raw/`:

```text
ne_10m_admin_0_map_units.geojson
ne_10m_admin_0_countries.geojson
ne_50m_admin_0_map_units.geojson
ne_50m_admin_0_countries.geojson
```

If those files are absent, missing islands are left hidden rather than replaced with fake rectangles.

`backend/data/base/manual_regions_1933.json` is kept as command-parser metadata and aliases. It is not used as player-facing polygon geometry.

Generated regions are kept for engine experiments but are not shown as player-facing labels or borders.

Microstates without usable country polygons are shown as clickable point markers. Country labels are conservative geometry-based labels: compact countries can use short dynamic line labels, while wide or fragmented countries fall back to safer representative-point labels.

`backend/data/save/campaign_state.json` stores province ownership, divisions, visual orders, and events for the current campaign.

TODO: saves are currently global. Later they should be isolated per user/session/campaign, for example `backend/data/save/{campaignId}/campaign_state.json` or a database-backed `campaigns` table.

## Known Issues

- Province geometry is MVP/debug, not final.
- Admin-1 regions are modern administrative regions clipped to 1933 countries, not perfect historical 1933 states.
- Country labels are conservative dynamic geometry-based label lines/points, but not final HOI-style typography yet.
- Greenland and missing islands should come from Natural Earth Admin-0/map units later; fake rectangle fallback polygons are intentionally disabled.
- No AI judge, combat model, or full pathfinding yet.
- User/session-isolated saves are planned later.
